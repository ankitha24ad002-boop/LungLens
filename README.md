# LungLens AI

AI-assisted Chest X-ray Triage System.

## Features

- Chest X-ray upload interface
- AI prediction dashboard
- Disease confidence analysis
- Grad-CAM explainability
- Recommendation system
- Medical report generation
- Prediction history
- Model information dashboard

## Technology Stack

Frontend:
- Streamlit
- Python
- Custom CSS UI

AI:
- PyTorch
- CNN-based model
- Grad-CAM Explainability

Backend:
- FastAPI (integration pending)
